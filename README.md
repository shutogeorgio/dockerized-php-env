# Make Developing Environment For PHP By Docker
## Operated by MySQL and Nginx
## Prerequisites
- OS X
- Docker for Mac

## Install the module and work
- make sure the time zone at `php.ini` (default: Europe/London)
-  Please check if docker-compose is installed 
	`$ docker-compose version`
	
-  `cd` command to the directory which this module is installed 
- up the container by `$ docker-compose up -d`
- Now you can access to your `localhost:8080`:
- For stoppting the container `$ docker-compose stop`
